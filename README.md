# My Portfolio Website
This is my personal portfolio website created using *only HTML*.  
It showcases my basic information, skills, projects, and contact details.

# About the Project
The purpose of this portfolio is to:
- Practice HTML fundamentals
- Present my personal and academic details
- Create an online identity as a beginner web developer

This project does *not use CSS or JavaScript* and is built purely with HTML tags.

# Technologies Used
- HTML5

##  Features
- Profile information
- Education details
- Skills list
- Project section
- Contact information
- Simple layout using HTML elements like:
  - headings
  - paragraphs
  - lists
  - forms
  - hr and br tags

##  How to View the Project
1. Download or clone this repository
2. Open the index.html file in any web browser
